- **Monorepo 管理**：采用 `pnpm workspaces` 进行包管理，使用 `Turbo` 实现任务编排与缓存加速（见 `turbo.json`）。
- **后端架构 (`apps/nestjs-server`)**：基于 NestJS 11，采用模块化设计（Auth, User, Health），通过 `APP_GUARD/INTERCEPTOR` 实现全局 JWT 鉴权、限流及日志转换，使用 Prisma + SQLite 作为数据层。
- **前端架构 (`apps/web`)**：基于 Vite + React 19，采用 `react-router` 路由与 `TanStack Query` 状态管理，UI 层面集成 `shadcn/ui` (Radix UI + Tailwind CSS)。
- **共享层 (`packages/shared`)**：通过 `tsdown` 构建，利用 `zod` 定义前后端统一的 Schema 校验规则与 TypeScript 类型，确保端到端类型安全。
